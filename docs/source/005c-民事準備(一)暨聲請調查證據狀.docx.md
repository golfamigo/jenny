2. **原告發現系爭房屋漏水瑕疵並未逾民法承攬章節規定之瑕疵發現期間。**  
1) 按「第四百九十三條至第四百九十五條所規定定作人之權利，如其瑕疵自工作交付後經過一年始發見者，不得主張。工作依其性質無須交付者，前項一年之期間，自工作完成時起算。」、「工作為建築物或其他土地上之工作物或為此等工作物之重大之修繕者，前條所定之期限，延為五年。」，民法第498條、第499條分別定有明文。  
2) 次按臺灣高等法院106年度上字第252號民事判決意旨：「按民法第493條至第495條所規定定作人之權利，如其瑕疵自工作交付後經過1年始發見者，不得主張。民法第498條第1項定有明文。又民法第499條雖規定工作為建築物或其他土地上之工作物或為此等工作物之重大之修繕者，前條所定之期限，延為5年，然所謂建築物係指建築物本身結構體而言，消防等設備依建築法第10條規定乃建築物設備，如未因附合而成為不動產之重要成分，尚無民法第499條之適用，最高法院75年度台上字第1906號、92年度台上字第1767號裁判均同此見解，可資參照。易言之，**若承攬之工作為木工、水電、磁磚舖貼或僅為小部分之泥作工程，應認已附合於房屋結構體，成為不動產之重要成分，屬所承攬之工作為建築物，而有民法第499條規定之5年瑕疵發現期間之適用**。」。  
3) 查本件相關承攬工程，被告吳政倫工程總價高達233萬元（按：最後收取60萬元），被告睿創空間設計有限公司工程總價高達385萬元，而工程項目包括隔間、壁面拆除，地磚壁磚重新鋪設、地面防水工程等，涉及系爭房屋結構層重新施作，且因相關泥作、木工、水電、磁磚鋪貼等工程已附合於房屋結構體，成為不動產之重要成分，屬所承攬之工作為建築物，當應有民法第499條規定之5年瑕疵發現期間之適用。是以，原告尚未逾法定瑕疵發現期間。  
4) 再者，原告與被告吳政倫簽約後，因被告吳政倫施作品質不佳，原告約於112年2月底與被告吳政倫合意解約，在原告與後手即被告睿創空間設計有限公司於112年3月11日簽約前，被告吳政倫始交還系爭房屋。而被告睿創空間設計有限公司開始施作後，原告於113年1月28日即發現有壁面有漏水問題（原證19），當日並有請大樓總幹事張克華陪同查看。故即便依民法第498條規定之一年期間，原告亦未逾越該瑕疵發現期間。  
3. **被告抗辯系爭房屋漏水係因地震造成裂縫云云，然渠等並未依法舉證，其辯詞當無可採。**  
1) 按當事人主張有利於己之事實者，就其事實有舉證之責任；民事訴訟法第277條前段定有明文。另主張常態事實者，就其事實無庸舉證，主張變態事實者，應就變態事實負舉證責任，此為舉證責任分擔原則。又原告（即被上訴人）對於自己主張之事實已盡證明之責後，被告（即上訴人）對其主張，如抗辯其不實，惟並無確實證明方法或僅以空言爭執者，當然認定其抗辯事實之非真正，而應為被告不利益之裁判（最高法院18年上字第1679、2855號判例參照）。  
2) 查被告抗辯系爭房屋漏水係因地震造成裂縫云云，然被告並未舉證以實其說，其抗辯之詞未盡舉證責任當無可採。且事實上，根據原證11鑑定報告，系爭房屋漏水原因包括「主臥室廁所」地坪有漏水、「主臥室廁所」門檻有滲漏水、「主臥室廁所」與窗框間的縫隙過大，使用會有明顯滲透水至主臥室內部房間牆壁等（請參見原證11第7頁），均與自然地震所造成房屋頂樓平台或外牆之裂縫無關，足證被告所辯純屬虛妄。  
4. **聲請調查證據：謹聲請由社團法人台灣營建防水技術協進會（地址：台北市中正區北平東路24號5樓之6。電話：(02)2395-6763）為本件漏水鑑定單位。鑑定事項如下：**

3) 承上，如欲修復上開滲漏水，系爭房屋的修復方法、修復項目、修復費用（材料、工資）、修復期間各為何？是否與原證11所示台灣防水工程技術協進會漏水原因鑑識報告書所認定者相符？
