承攬 / 保固範圍主張

原證4 原證5 原證11

核心主張

  • 前手吳政倫的承攬範圍本就包括主臥浴室隔間、地磚、防水與相關泥作工程。

Point 2

瑕疵發現期間主張

民法 499 2024-01-28 / 2024-01-30

核心主張

  • 本件工程已附合於建築物，應有民法第 499 條 5 年瑕疵發現期間之適用。
  • 頁面整理採「2024-01-28 察覺壁癌、2024-01-30 為起訴狀版本記載的發現日期」；即使採較早的 2024-01-28 起算，2024-11-29 起訴也可主張未逾越一般期間。

用途

主要是先擋掉對方若主張「太晚發現、太晚起訴」的程序型抗辯，讓法院能進到實體漏水責任判斷。

Point 4

鑑定事項主張

漏水原因 修復方法 修復期間可住性

目前主張方向

  • 是否有滲漏水、位置在哪裡、源頭為何。
  • 若要修復，方法、項目、費用與期間為何。
  • 修復期間能否正常居住，或是否必須另覓住處。

還應補問的技術點

  • 紅磚牆 / 原牆 / 氣密窗交界之異材質轉角工法。
  • 底層是否應拆到底重做。
  • 木地板浸潤與漏水路徑的因果關係。
  • 前手本體瑕疵與後手防水收邊瑕疵如何切分。

為何漏水因果要具體落點

本件不能只抽象地問「有沒有漏水」或「工法有沒有問題」，而要把因果關係落到具體位置與具體缺失。也就是要問清楚：水到底是從紅磚牆 / 原牆 / 氣密窗交界、窗框與牆面接縫、門檻，還是地坪進入；又是因為少了哪一個施工步驟才會進水，例如未填縫、未做防水包覆、未補強收邊、未拆到底重做，或基底 / 防水層處理不足。只有把「水從哪裡進」和「因為少做哪一步才進」這兩件事問清楚，法院才比較容易判斷前手、後手或兩者共同責任，也才能往後連到修復範圍、修復費用與木地板等衍生損害的因果鏈。

Point 5

訴訟結構評估

傾向共同被告 不是自動連帶

目前結論

  • 以本案材料看，單純維持「後手先位、前手備位」並不理想。
  • `004-民事陳報狀` 補充的原始程序理由是：兩被告對同一漏水紛爭有互相推諉責任情形，因此原告先採主觀預備合併，希望在同一程序內一次釐清。
  • 較有利的方向，是把兩被告放在同一程序架構下，讓法院依鑑定結果判斷前手、後手或兩者共同責任。
  • 原因在於本案不像乾淨的二選一，而是前手本體施工與後手重作 / 保固責任可能重疊。

要一起講清楚的限制

  • 共同被告不等於當然連帶賠償。
  • 正式變更請求結構，仍要看目前程序階段與律師是否能用同一基礎事實承接。
  • `004-民事陳報狀` 曾援引最高法院 94 年度台上字第 283 號、94 年度台抗字第 980 號、94 年度台上字第 1078 號，作為原先主觀預備合併的程序支撐。
  • 法律視角上，程序面看共同訴訟與變更追加，實體面看承攬瑕疵與侵權責任如何銜接。
  • 完整分析放在時間軸頁的「程序與訴訟結構」段落，本頁只保留摘要結論。
