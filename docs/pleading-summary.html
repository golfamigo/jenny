書狀重點整理

這一頁把 [民事準備(一)暨聲請調查證據狀] 的重點換成較好閱讀的版本。 目的是讓相關人快速看懂目前主要在打什麼，不必直接打開 GitHub 上的 Markdown 原稿。

Point 1

承攬 / 保固範圍主張

原證4 原證5 原證11

核心主張

  • 前手吳政倫的承攬範圍本就包括主臥浴室隔間、地磚、防水與相關泥作工程。
  • 後手睿創接手時，曾表示會重做防水、處理先前漏水問題，並出具 10 年防水保固。
  • 因此，不能只因「新砌隔間牆打底工資已扣除」就全面切斷睿創的保固責任。

對應證據

  • E01 第一次鑑定書
  • E08 睿創估價單
  • E11 阿倫工程內容與睿創保固
Point 2

瑕疵發現期間主張

民法 499 2024-01-28

核心主張

  • 本件工程已附合於建築物，應有民法第 499 條 5 年瑕疵發現期間之適用。
  • 即使退一步看，原告於 2024-01-28 發現漏水，2024-11-29 起訴，也未逾越一般期間。

用途

主要是先擋掉對方若主張「太晚發現、太晚起訴」的程序型抗辯，讓法院能進到實體漏水責任判斷。

Point 3

地震抗辯主張

對方抗辯 原證11

核心主張

  • 對方稱漏水來自 0403 地震造成裂縫，但未依法提出足夠舉證。
  • 現有材料反而指向主臥浴室地坪、門檻、窗框接縫與交界處理缺失。

對應證據

  • E01 第一次鑑定書
  • E02 窗框邊空洞
  • E05 噴水穿透影片
  • E06 / E07 異材質交界與裂縫照片
Point 4

鑑定事項主張

漏水原因 修復方法 修復期間可住性

目前主張方向

  • 是否有滲漏水、位置在哪裡、源頭為何。
  • 若要修復，方法、項目、費用與期間為何。
  • 修復期間能否正常居住，或是否必須另覓住處。

還應補問的技術點

  • 紅磚牆 / 原牆 / 氣密窗交界之異材質轉角工法。
  • 底層是否應拆到底重做。
  • 木地板浸潤與漏水路徑的因果關係。
  • 前手本體瑕疵與後手防水收邊瑕疵如何切分。
