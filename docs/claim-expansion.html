訴之擴張討論頁

本頁是供原告與律師盤點所有可能求償項目、法律基礎、現有證據與缺失候補的工作頁。重點是先確保修繕金、鑑定費、木地板與附帶裝修損害足額回復，再評估其他擴張項目。

保本主軸

這一區應優先鎖定可直接回復原狀或與漏水損害有明確因果的支出，先把修復本體與附隨損害做足。

  • 漏水本體修復費。
  • 鑑定費與必要檢測費。
  • 木地板損害。
  • 附帶裝修復原。
  • 恢復原有居住品質所必要的直接支出。

策略擴張項

這一區建議以 5 大類整理，但可拆成 8 個具體子項來判斷是否納入，避免金額、法律基礎與證據成熟度混在一起。

  • 遷居與住宿。
  • 搬運與倉儲。
  • 除霉 / 除蟲 / 清潔。
  • 居住空間縮減或部分失能損害。
  • 身心損害（健康損害、精神慰撫金）。

具體子項可再拆為 8 項：修復期間遷居費、住宿費、搬運費、倉儲費、除霉 / 除蟲 / 清潔費、居住空間縮減或部分失能損害、健康損害、精神慰撫金。

民法第 493 條

承攬工作有瑕疵時，得請求修補、解除或減少報酬，是漏水本體修復與瑕疵責任的起點。

民法第 495 條

承攬人不於相當期間補正時，定作人得請求賠償，因此可承接修復費與另行發包費用。

民法第 213 條

損害賠償以回復原狀為原則，木地板、附帶裝修與必要復原支出可放在這條路徑下整理。

民法第 216 條、227 條

可主張所受損害及所失利益內的直接支出，並以債務不履行補充承接修復期間衍生費用。

民法第 195 條、227-1 條

主要對應健康損害與精神慰撫金等非財產上損害；財產性支出仍以 213、216、227 路徑整理較一致。

民訴法第 255 條

若本次新增請求仍基於同一漏水紛爭事實，律師可優先檢視「請求基礎事實同一」與「擴張應受判決事項之聲明」路徑，處理訴之擴張或追加列項。

保本主軸
漏水本體修復費
浴室、窗框、牆面、門檻、地坪及漏水路徑本身的拆作與重作。
 民法 493、495、213、216、227 若承攬工程及防水處理存在瑕疵，原告得請求回復原狀所需修復費用。 第一次鑑定書、噴水穿透影片、窗框空洞與裂縫照片、既有估價單。 最新修繕估價單、最新鑑定或複估。 以最新修繕估價單、拆作範圍與工法明細加總；目前金額待估。 建議由民法 493、495 搭配 213、216、227 作為主請求架構，先以最新修繕估價確定本體工項，再由律師依民訴 255 條評估直接擴張金額或改以分項列明。
保本主軸
鑑定費與必要檢測費
包含法院鑑定費、紅外線檢測、灑水測試或必要技術檢測。
 民法 213、216、227、495 為釐清漏水原因與修復方法所支出的必要費用，屬回復原狀及損害賠償範圍。 法院函、土木技師公會函、既有鑑定相關文件。 實際繳費收據、後續必要檢測單據。 依已繳鑑定費與檢測費單據列計；目前部分項目待單據。 建議先把已實際支出的法院鑑定費列為確定額，再由律師判斷是否把後續必要檢測費一併列入，並以收據或繳費證明承接 213、216、227 的必要費用路徑。
保本主軸
木地板損害
主臥或受潮區域木地板拆除、重鋪、材料與工資。
 民法 213、216、227、495 木地板因漏水受潮、膨脹、發霉或失去原有使用功能，屬直接損害。 木板拆開照片、潮濕蟲與內部異常照片、現況照片。 木地板拆除 / 重鋪 / 材料 / 工資單據、因果補充鑑定。 依拆除重鋪估價、坪數、材料等級與工資計算；目前待估。 建議直接獨立列為保本主軸項目，不併入本體修復概算；由律師以 213、216、227 搭配 495 主張，並要求估價單與因果說明把受潮、膨脹、發霉與漏水路徑接起來。
保本主軸
附帶裝修復原
受影響牆面、油漆、收邊、門框、踢腳板或必要拆作後的復原。
 民法 213、216、227、495 修復漏水通常必然伴隨局部裝修拆除與重作，若不復原即無法回到原使用狀態。 現況照片、第一次鑑定書、既有估價單、進場與牆面層次照片。 最新復原估價單、必要拆作說明。 依復原工項明細加總；目前待估。 建議與本體修復並列，但在書狀上維持獨立項目，讓律師可明確主張「若不復原即無法回到原使用狀態」，避免被對方切割成非必要美化支出。
保本主軸
恢復原有居住品質所必要的直接支出
臨時封護、除濕、保護性拆除、必要替代處置等直接支出，不含專業除霉 / 除蟲 / 清潔費。
 民法 213、216、227 只要屬於為避免損害擴大或回復正常居住所必要支出，即可列為直接損害。 現況照片、對話紀錄中長期不能正常使用描述。 除濕設備租借、臨時處置、保護工程單據。 依實際發票或收據列計；目前多數待單據。 建議先限縮為臨時防護、除濕、保護性拆除與必要替代處置，讓律師以 213、216、227 直接整理成回復原狀必要支出，並與後面的專業除霉 / 除蟲 / 清潔費嚴格切開。
策略擴張項
修復期間遷居 / 住宿 / 搬運 / 倉儲
包含修繕期間另覓住處及物品移置成本。
 民法 213、216、227 若修復工程客觀上使原居所無法正常居住，遷居與搬運為可預見之必要支出。 現況照片、不能正常使用敘述、鑑定程序文件。 住宿搬運倉儲單據、修復期間不可居住的鑑定承接。 依實支住宿日數、搬運報價、倉儲月費計算；目前待單據、待鑑定。 若目前不可居住期間尚未被鑑定鎖定，建議律師先列原因事實與法律基礎，金額採待估或後補單據；若鑑定已可承接期間與範圍，再改為具體數額擴張較穩。
策略擴張項
除霉 / 除蟲 / 清潔費
針對潮濕、霉菌、蟲體與污染區域的專業處理。
 民法 213、216、227 若潮濕與漏水已引發霉斑、蟲害與衛生風險，該處理支出與漏水有直接關聯。 潮濕蟲與木板照片、現況照片。 除霉除蟲清潔與暫時處置單據、專業清潔報價。 依清潔公司報價或實際發票列計；目前待補單據。 建議獨立成項，並由律師先判斷要作為保本直接支出的延伸，或列為策略擴張項；無論採哪一路徑，都應先補報價、發票或實支單據，避免只停在照片與敘述。
策略擴張項
居住空間縮減或部分失能損害
例如房間、浴室長期無法完整使用，造成實際居住功能受限。
 民法 213、216、227 若部分空間長期失去原功能，可主張居住利益受損，但需具體量化基礎。 現況照片、長期不能正常使用的對話敘述。 修復期間不可居住的鑑定承接、空間受限期間整理表。 依受限空間坪數、期間、替代支出或使用減損方式估算；目前待鑑定。 建議先由律師確認此項要不要獨立請求；若要主張，最好先建立受限空間、期間與替代成本三件事，並以鑑定或時間整理表支撐，不宜只有抽象「很困擾」敘述。
策略擴張項
健康損害
若黴菌、潮濕或蟲害造成具體身體不適或就醫。
 民法 195、227-1 需證明漏水環境與身體不適間有醫療紀錄或相當因果連結，方適合列入。 目前以居住環境惡化事實為主，醫療證據不足。 醫療紀錄、診斷證明、用藥或就醫收據。 依醫療支出實支列計，其他損害待醫證支持；目前待單據。 建議律師先用民法 195、227-1 檢視是否達到可主張門檻；若目前只有環境惡化、沒有醫療紀錄，較穩的做法是先保留事實與補件方向，不急著列具體金額。
策略擴張項
精神慰撫金
以長期居住受擾、生活品質嚴重受損為核心事實。
 民法 195、227-1 若可具體說明長期漏水、霉味、房間失能與生活重大不便，可評估非財產上損害。 現況照片、對話敘述、長期無法正常使用事實。 生活影響整理、必要時輔助醫療或諮商資料。 依個案情節斟酌，通常待估。 建議律師先判斷本次是否把精神慰撫金作為附加請求而非主軸，並檢視要與健康損害分列或擇一強化；若證據仍以生活受擾為主，可先保留事實與法條路徑，避免過早定額。

律師討論

律師決策點

主金額 列項方式 先列事實或先列金額

程序上，若本次新增求償仍基於同一漏水紛爭事實，律師可優先檢視民事訴訟法第 255 條第 1 項第 2、3 款；實體上則以民法 493、495、213、216、227 為財產損害主軸，195、227-1 作為健康損害與精神慰撫金的補充路徑。

1. 是否以最新修繕估價與木地板復原費作為本次擴張主金額

律法實務建議：先把最新修繕估價、木地板重鋪估價、附帶裝修復原估價整理成三個可獨立計算的區塊，再由律師決定是以整體修繕總額主張，或採分項列明方式主張。

  • 法條抓手：民法 493、495、213、216、227。
  • 先做事項：補齊最新估價單、工法範圍、拆作與復原明細。
  • 程序建議：若仍屬同一漏水紛爭事實，可朝民訴 255 條的擴張路徑處理。

2. 木地板與附帶裝修是否在本次擴張中獨立列項

律法實務建議：這兩項已屬保本主軸，宜在書狀中獨立列項，避免被對方主張只是本體修復的附隨費用，或被法院吸收到單一修繕金內。

  • 法條抓手：民法 213、216、227，必要時補 495。
  • 先做事項：把木地板與附帶裝修的估價、範圍、照片、因果說明分開整理。
  • 實施方式：主張「不復原即未回到原使用狀態」，而非僅作美化支出。

3. 遷居 / 住宿費是否先列原因事實，待鑑定後補金額

律法實務建議：如果目前尚未由鑑定明確指出不可居住期間與修復天數，先列原因事實與法條基礎通常比先硬列金額更穩，之後再補單據與具體期間。

  • 法條抓手：民法 213、216、227。
  • 先做事項：補住宿、搬運、倉儲單據，並確認鑑定是否會就可居住性表意見。
  • 實施方式：先列事實保留後補金額，或在鑑定後一次擴張到具體數額。

4. 精神慰撫金與健康損害本次是否主張

律法實務建議：這兩項不宜成為本次主軸。若醫療與生活影響資料尚未成形，先保留事實與補件方向，通常比現在就列高額非財產損害更穩。

  • 法條抓手：民法 195、227-1。
  • 先做事項：蒐集醫療紀錄、診斷證明、用藥收據或生活影響整理。
  • 實施方式：可先主張健康或精神其中一項，避免兩者同時列而證據不足。

5. 是否需補強不可居住期間與居住空間縮減的量化基礎

律法實務建議：若要主張遷居費、部分失能或空間縮減損害，律師應先決定量化路徑，不然容易只有受損敘述，沒有可被法院採認的計算基礎。

  • 法條抓手：民法 213、216、227。
  • 先做事項：整理受限空間、期間、替代支出、每日或每月成本。
  • 實施方式：可採鑑定量化、實支量化，或兩者並行後擇較穩版本主張。

6. 恢復原有居住品質所必要的直接支出要併入主項還是獨立列項

律法實務建議：這一項的關鍵不是有沒有，而是邊界。若不先切清楚，很容易與除霉 / 除蟲 / 清潔費、附帶裝修或本體修復互相重疊。

  • 法條抓手：民法 213、216、227。
  • 先做事項：把臨時封護、除濕、保護性拆除、替代處置獨立列表。
  • 實施方式：若律師偏好清楚可讀，可列為獨立次項；若偏好精簡，可併入回復原狀必要費用。
