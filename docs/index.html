  • 細項一：起訴狀附件修繕費 368,898 元（179,626 元＋189,272 元）。

民事準備(一)暨聲請調查證據狀

  • 已主張前手施工範圍、後手重作 / 保固責任。
  • 已聲請鑑定位置、原因、修復費用、修復期間與可居住性。

目前程序焦點

  • 共同被告結構。
  • 補充鑑定題目。
  • 訴之擴張項目與金額路徑。
第二優先區塊

核心爭點摘要

前手責任 後手保固 漏水因果 未驗收

責任切分

前手本體施工責任，與後手重作、防水、保固責任，要在同一頁先被看清楚。

因果落點

漏水因果需落到窗框、門檻、地坪與異材質交界，而不是停留在抽象的「牆面滲水」。

程序語言

目前主張已經進到鑑定位置、原因、修復費用、修復期間與可居住性，不是只有單純查漏水。

抗辯回應

入住與付款不等於驗收完成，後手若主張已驗收，仍須回到未完工與未驗收脈絡處理。

第三優先區塊

待辦清單

待律師決策 待原告補件

待律師討論 / 決策

  • 是否改列共同被告。
  • 訴之擴張主軸是否先鎖定修繕、鑑定、木地板、附帶裝修。
  • 遷居 / 住宿 / 搬運 / 倉儲先列事實還是先列金額。
  • 健康損害與精神慰撫金本次是否主張。
  • 鑑定事項補充到哪一層。

待原告補件 / 提供證據

  • 最新修繕估價單。
  • 木地板拆除 / 重鋪 / 材料 / 工資單據。
  • 法院鑑定費與必要檢測費收據。
  • 住宿 / 搬運 / 倉儲 / 除霉除蟲清潔單據。
  • 醫療紀錄與診斷證明。
