# jenny

這個 repo 主要提供案件資料的可瀏覽閱讀版。

## 入口

- GitHub Pages 站點啟用後，首頁會是 `docs/index.html`
- 新增頁面：`docs/timeline.html`，整理案件時間軸與核心爭點
- 若尚未啟用 Pages，可先從 repo 內查看：
  - `docs/index.html`
  - `docs/timeline.html`
  - `docs/source/證據目錄版_圖文對話定位.md`
  - `docs/source/圖文對話整理_依截圖定位.md`

## 結構

- `docs/index.html`：圖文閱讀版
- `docs/timeline.html`：時間軸 + 爭點清單
- `docs/assets/`：圖片與影片
- `docs/source/`：原始整理文字檔
